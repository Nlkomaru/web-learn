## このアプリケーションの概要
「Web Learn」という、Web開発を学ぶためのアプリケーションです。
page部分とslide部分に分かれています。

## 主な機能
- page部分
  - ユーザーが簡単にSlideにアクセスできる。
- slide部分
  - Slidev世利用したスライド作成が可能



## 主な技術スタック
### Pages
- フレームワーク: React Router v7
- 言語: TypeScript
- パッケージマネージャー: pnpm
- Headless UIライブラリ: Ark UI
- UIライブラリ & デザインシステム: Chakra UI
- CSSライブラリ: panda css
- state管理: jotai with hash
